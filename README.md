# AssignmentPro - Academic Assignments Platform

A professional platform connecting students with academic writers for assignment help.

## Features

- **Dual Account System**: Separate accounts for clients (students) and freelancers (writers)
- **Order Management**: Complete workflow from order placement to delivery
- **Payment Integration**: Support for PayPal, M-Pesa, and credit cards
- **Secure Authentication**: User registration and login system
- **Responsive Design**: Works on all devices
- **File Upload**: Clients can upload assignment files
- **Order Tracking**: Real-time status updates

## Account Types

### Client Accounts (Students)
- Place assignment orders
- Upload assignment files
- Track order progress
- Make payments
- Rate and review writers

### Freelancer Accounts (Writers)
- Browse available assignments
- Bid on assignments
- Upload completed work
- Manage earnings
- Build reputation

## Setup Instructions

1. Clone or download this repository
2. Open `index.html` in a web browser
3. The application uses localStorage to simulate a database

## Pages

- `index.html` - Homepage with authentication
- `dashboard.html` - Client dashboard
- `freelancer-dashboard.html` - Freelancer dashboard
- `order-management.html` - Order management
- `payment.html` - Payment processing
- `profile.html` - User profile management

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript (ES6+)
- Font Awesome Icons
- LocalStorage for data persistence
